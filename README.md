# pyHive
- pyHive is a python3 implementation of the PHP CoinHive API class

###Usage

```python
coinhive = CoinHiveAPI("YOUR_SECRET_KEY")
stats = coinhive.get("/stats/site")
print(stats.get("history"))
```
